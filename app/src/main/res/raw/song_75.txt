<h>75. SONDÔ MFE A BETAYA KPWAAN</h>
<br> </br>
<t>Tune : Safely through another week. Basa 256 - Bafia 11</t>
<v>1</v>
		Sondô mfe a betaya kpwaan,
		Zambe, Ô nga ba'ale ma;
		M'aye jeñe mfefé ngu
		M'azu wô'ô nda jôé,
		Mbamba môs, a dañe mvaé,    } (bis)
		Ndem y'awo'an ya nnôm éto.  }
<v>2</v>
		M'azu ja'é Wo mvame jôé.
		Amu Yésus, Tate wom;
		Liti'i na, W'ajibi ma.
		Va'a m'ôson, a mam abé;
		Vôm m'atyelé nleme yôp,   } (bis)
		M'azu wo'an be wo, A Tat  }
<v>3</v>
		M'azu kañe Wo nda jôé,
		Vôm Ô too melu mese;
		Kui'me mis me nlem, A Tat,
		M'aye yen étôto'o jôé;
		Vôme nyô m'atye'e bidi,    } (bis)
		M'aye di bie nlam ya yôp.  }
<v>4</v>
		M'azu wô'ô Benya mejôô,
		Mme m'adutu bô bese;
		Môse wu m'ayene mvaé,
		Mfefé ngul ô ntoo ma nlem;
		Sond'ase a bo' ana,  } (bis)
		Akekui m'ake be wo.   }
<br> </br>
<a>Rev. Alvert Irwin Good</a>
